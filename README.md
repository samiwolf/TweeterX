## Requirements
Node Version 18.10.0

Ionic CLI Version 7.1.1
## Installation

Install Ionic and dependencies with npm

```bash
npm install -g @ionic/cli
npm install
```

Serve in localhost

```bash
ionic serve
```
Serve in localhost with specific port

```bash
ionic serve --port=PORT_NUMBER
```
Build for production

```bash
ionic build --prod
```
